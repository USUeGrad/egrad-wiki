What is eGrad?

eGrad is the simplest Student Management System for Higher Education. It provides an interface to easily manage applications and student matriculations. With a "one-stop-shop" design, it gets data from central repositories (e.g. Ellucian BannerÂ®, Box), reducing the amount of data duplication, allowing you to add an extra data layer to aid with collection and reporting of information.

Features

Program Admissions Management

eGrad allows students to file applications into each of your programs. Paired with a Central Authentication System (CAS), your students can easily login to eGrad, fill-out forms and upload files that can be reviewed by your program coordinators.

eGrad allows you to filter, organize, and score applications against a customizable rubric. Also, if required, it can be further sent to a pre-defined committee to review each application.
